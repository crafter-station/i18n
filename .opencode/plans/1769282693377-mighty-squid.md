# Fix: STT Not Starting / No Audio

## Problem Analysis

From the screenshot and user confirmation (still broken after unmuting):
1. WebSocket to ElevenLabs (`realtime?model_id=scribe...`) shows in Network tab
2. No STT indicator (green dot) appears
3. The WebSocket is likely **failing authentication**

### Root Cause: WebSocket Authentication Failure

ElevenLabs STT WebSocket requires authentication. Looking at the API docs:
- **Header auth**: `xi-api-key` header - **NOT possible in browser WebSocket**
- **Query param**: `token` - requires a **signed token**, not raw API key

The current code passes the raw API key as `token` param:
```typescript
wsUrl.searchParams.set("token", apiKey);  // WRONG - needs signed token
```

But ElevenLabs expects a **single-use signed token** generated from their token endpoint, not the raw API key.

## Solution

Generate a proper signed token from ElevenLabs for browser WebSocket authentication.

### ElevenLabs Token Generation

According to the ElevenLabs docs, tokens for browser WebSocket auth can be generated from:
```
POST https://api.elevenlabs.io/v1/text-to-speech/create-previews/create-token
```

Or for STT specifically, we may need to check if there's a dedicated endpoint.

**Alternative**: Check if ElevenLabs allows the API key directly via a different query param name.

```typescript
// Remove these imports

// Add back Daily.co transcription on join:
useEffect(() => {
  if (!daily) return;

  const join = async () => {
    try {
      await daily.join({ url: roomUrl, token });
      
      // Start Daily.co transcription (uses Deepgram)
      await daily.startTranscription({
        language: "multi",
        model: "nova-2",
        punctuate: true,
        profanity_filter: false,
      });
      
      setIsJoining(false);
    } catch (error) {
      console.error("Failed to join call:", error);
    }
  };

  join();

  return () => {
    daily.stopTranscription();
    daily.leave();
  };
}, [daily, roomUrl, token]);

// Change from app-message to transcription-message:
useDailyEvent("transcription-message", async (event) => {
  if (!event?.text) return;
  
  const isFinal = event.rawResponse?.is_final ?? true;
  if (!isFinal) return;
  
  const { text, participantId } = event;
  if (participantId === localParticipant?.session_id) return;
  
  // ... translate and TTS ...
});
```
